# 方块合合合

竖屏 H5 数字合成闯关小游戏 MVP。玩家控制下落方块左右移动或快速落下，在 5x8 棋盘内触发 2048 风格合成，并用有限步数达成目标数字。

## 技术栈

- Vite
- TypeScript
- Phaser 3
- LocalStorage 本地存档

## 运行

```bash
pnpm install
pnpm dev
```

## 验证

```bash
pnpm lint
pnpm test
```

## MVP 内容

- 首页、关卡选择、挑战主界面、无尽模式、通关/失败弹窗
- 10 个残局挑战关卡
- 5 列 x 8 行棋盘、2048 经典暖色阶梯配色、高级方块 Glow
- 1-2 步触发多米诺式连续合并的残局设计
- 挑战模式步数三星评价、无尽模式最高分、本地进度存档
- 通关炫耀文案、失败看视频加步复活、锤子道具激励广告 Mock
- 抖音小游戏平台能力 Mock：广告、分享、震动、登录入口
